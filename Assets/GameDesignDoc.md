## 1. 世界観

言葉が力を失った世界。文明は崩壊し、ゾンビが闊歩する廃墟で、
ただ一人の生存者は『漢字を組み合わせて万物を具現化する』能力に覚醒した。
グリッド状に区切られた廃墟を1マスずつ慎重に進み、
散らばる漢字パーツを集めて武器を合成し、生き延びろ。

|------|------|
| ジャンル | グリッドベース・ターン制サバイバルパズル |

| テーマ | 漢字合成による武器創造 |
| 移動システム | 1マス単位のグリッド移動（物理演算なし） |
| ターン制 | プレイヤー行動 → 敵ターン → 次のターン |
| 画面構成 | 左70%: ゲーム画面 / 右30%: インベントリ&合成UI |
| コアループ | 探索 → 漢字パーツ収集 → 漢字合成 → 戦闘 |
