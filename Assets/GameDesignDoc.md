## 1. 世界観

言葉が力を失った世界。文明は崩壊し、ゾンビが闊歩する廃墟で、
ただ一人の生存者は『漢字を組み合わせて万物を具現化する』能力に覚醒した。
フィールドに散らばる漢字パーツを拾い集め、合成して武器を創り出し、生き延びろ。

| 項目 | 内容 |
|------|------|
| ジャンル | 見下ろし型アクションサバイバル |
| 視点 | 2D トップダウン（Orthographic） |
| テーマ | 漢字合成による武器創造 |
| 描画方式 | SpriteRenderer + TextMesh（2D） |
| 物理エンジン | Physics2D（Rigidbody2D / Collider2D） |
| コアループ | 探索 → 漢字パーツ収集 → 漢字合成 → 戦闘 |

## 2. 操作方法

| キー / 入力 | アクション |
|------------|-----------|
| W / A / S / D | 移動（上左下右） |
| マウス移動 | プレイヤーの向き（照準） |
| 左クリック | 漢字武器で射撃 / 攻撃 |
| C | 漢字合成の実行 |
| 接近（トリガー） | 漢字パーツ自動取得 |
